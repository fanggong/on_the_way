# On The Way Web (v0.4.0)

React + TypeScript + Semi Design web client.

## Scope

- Login / Register
- First-login Profile Setup
- App layout with 8 system modules
- System source management (core source, sync policy, backfill job)
- Health module pages:
  - Connector Config
  - Domain Query
  - Mart View
- 7 non-health modules as `开发中` placeholders

## Run

```bash
npm install
npm run dev
```

Environment:

```bash
cp .env.example .env
# set VITE_API_BASE_URL if needed
```
